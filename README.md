Job -- a configuration of what Jenkins has to do... It specifies from where it should get code, when to run build (trigger, cron, webhook), on which agent/node, which steps to take and how to archive artifacts. Job is a template whilist build is specific run of this template... 
Build -- single run of specific task in Jenkins. It has a number (#123), staus (Fail/Success/Aborted, time, logs, artifacts, and enviromental variables, tests' results.
Pipeline -- a way of defining job as a code...
Stage -- logical step of Jenkins pipline which might include steps such as checkout/test/artifacts...
Step -- single action in Jenkins pipeline sh/checkoput/archiveArtifacts...
Agent -- place where pipeline is being executed.. 
Workspace -- a drectory with repository on executing agent...
Artifact -- file resulting from Jenkins pipeline..
